15번 문제

가변 위치 인수(*args) — 최댓값
max_of(*args) 함수를 작성하세요. 전달된 값 중 최댓값을 반환하세요. (인자가 없으면 None)

힌트 / 출력
호출 예시
print(max_of(3, 7, 2))
print(max_of())
출력 예시
7
None
정답 코드 보기
정답
def max_of(*args):
    if len(args) == 0:
        return None
    m = args[0]
    for x in args[1:]:
        if x > m:
            m = x
    return m

print(max_of(3, 7, 2))
print(max_of())

    result = []
    for x in nums:
