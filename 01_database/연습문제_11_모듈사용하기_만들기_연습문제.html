13번 문제

텍스트 파일 읽기 — .txt만 모아서 내용 리스트 만들기
현재 폴더에 t1.txt, t2.txt, note.md를 만들고, 현재 폴더에서 확장자가 txt인 파일만 읽어서 내용을 리스트로 모으세요.

힌트 / 출력
출력 예시
['A', 'B']
정답 코드 보기
정답
import os

with open('t1.txt','w',encoding='utf-8') as f:
    f.write('A')
with open('t2.txt','w',encoding='utf-8') as f:
    f.write('B')
with open('note.md','w',encoding='utf-8') as f:
    f.write('X')

files = []
for file_name in os.listdir():
    if file_name[-3:] == 'txt':
        with open(file_name, encoding='utf-8') as file:
            data = file.read()
            files.append(data)

print(sorted(files))

29번 문제

모듈 만들기 — Cal 클래스 작성
myfunc.py에 Cal 클래스를 작성하세요. 생성자에서 num1, num2를 저장하고, add()는 합, sub()는 차(num1-num2)를 반환해야 합니다.

-1

class Cal:
    def __init__(self, num1, num2):
        self.num1 = num1
        self.num2 = num2

    def add(self):
        return self.num1 + self.num2

    def sub(self):
        return self.num1 - self.num2

현재 폴더에 myfunc.py가 없으면, 최소한 add(a,b)가 있는 파일을 생성하세요. 그 다음 add(1,2)를 출력하세요. (힌트: exists + open/write)

힌트 / 출력
출력 예시
3
정답 코드 보기
정답
import os

if not os.path.exists('myfunc.py'):
    with open('myfunc.py', 'w', encoding='utf-8') as f:
        f.write('def add(a, b):\n    return a + b\n')

from myfunc import add
print(add(1, 2))

39번 문제

datetime + myfunc — 시간 문자열과 합 출력
datetime으로 2026-01-05를 만들고, myfunc.add(3,4) 결과와 함께 한 줄로 출력하세요. 형식: 2026-01-05: 7

힌트 / 출력
출력 예시
2026-01-05: 7
정답 코드 보기
정답
from datetime import datetime
import myfunc

fixed = datetime(2026, 1, 5)
print(f"{fixed.strftime('%Y-%m-%d')}: {myfunc.add(3, 4)}")

40번 문제

종합 — Cal 상속 + 날짜 포맷 함께 출력
MoreCal(Cal)에 mul()을 만들고, 2026.01.05와 곱셈 결과를 같이 출력하세요. 형식: 2026.01.05 20

힌트 / 출력
출력 예시
2026.01.05 20
정답 코드 보기
정답
from datetime import datetime
from myfunc import Cal

class MoreCal(Cal):
    def mul(self):
        return self.num1 * self.num2

fixed = datetime(2026, 1, 5)
dul = MoreCal(4, 5)
print(fixed.strftime('%Y.%m.%d'), dul.mul())
