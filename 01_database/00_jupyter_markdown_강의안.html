마크다운이란

마크다운(Markdown)은 글을 쉽고 빠르게 구조화해서 작성하기 위한 문법이다. Jupyter Notebook에서는 셀을 나눠서 문서와 코드를 분리한다.

  • Code Cell은 Python 코드를 실행하는 셀이다.
  • Markdown Cell은 설명, 제목, 문서 내용을 작성하는 셀이다.

셀 전환 & 실행

마크다운 셀 작성 흐름은 다음과 같다.

  1. 셀을 선택한다.
  2. Esc를 누른 뒤 M을 누르면 Markdown 셀로 바뀐다.
  3. 내용을 작성한다.
  4. Shift + Enter로 실행하면 렌더링되어 화면에 표시된다.
단축키 Esc → M은 마크다운 셀 전환, Esc → Y는 코드 셀 전환이다.

코드

인라인 코드

문장 중간에 코드를 표시할 때는 백틱 1개를 사용한다.

변수는 `x = 10` 처럼 정의한다.

코드 블록

여러 줄 코드를 표시할 때는 백틱 3개를 사용한다. 언어 이름을 쓰면 문법 강조가 적용된다.

```python
for i in range(5):
    print(i)
```
팁 python, sql, bash 같은 언어명을 붙이면 가독성이 좋아진다.

인용문

인용문은 >로 시작한다.

> 데이터 분석에서 가장 중요한 것은 질문이다.

구분선

수평 구분선은 ---로 만든다.

---

링크

링크는 [텍스트](URL) 형식이다.

[구글 바로가기](https://www.google.com)

이미지

이미지는 ![대체텍스트](이미지경로) 형식이다.

![그래프 예시](plot.png)
팁 로컬 파일 경로 또는 이미지 URL도 가능하다.

표

표는 |로 칸을 나누고, 헤더 아래에 구분선을 넣는다.

| 이름 | 나이 | 전공 |
|---|---|---|
| 철수 | 20 | 컴퓨터 |
| 영희 | 21 | 수학 |

정렬

| 왼쪽 | 가운데 | 오른쪽 |
|:---|:---:|---:|
| A | B | C |

체크박스

체크박스는 - [ ], 체크 완료는 - [x]이다.

- [x] 데이터 수집
- [ ] 전처리
- [ ] 모델 학습

수식(LaTeX)

인라인 수식

$y = ax + b$

블록 수식

$$
\sum_{i=1}^{n} x_i
$$
팁 통계/머신러닝/선형대수 노트 정리에 특히 유용하다.

주의사항

  • Markdown 셀에서는 Python 코드가 실행되지 않는다.

  • 문서 설명은 Markdown Cell에서 작성하는 것이 좋다.
