Jupyter Markdown 실습문제 30

각 문제는 “Markdown 셀”에서 직접 입력해보고, 렌더링 결과를 확인하는 실습이다.

실습 방법 문제

실습 방법

  1. Jupyter Notebook에서 새 셀을 만든다.
  2. Esc → M으로 Markdown 셀로 바꾼다.
  3. 문제의 “작성할 마크다운”을 그대로 입력한다(또는 요구사항을 만족하도록 작성한다).
  4. Shift + Enter로 렌더링 결과를 확인한다.
체크 포인트 “작성할 마크다운”은 정답 예시일 뿐이며, 같은 결과가 나오면 다른 표현도 정답이다.
