문제 95
95. sales 에서 가장 많이 팔린 값이 어느 위치(행,열)인지 np.argmax 를 이용해 찾아서 그 행과 열 인덱스를 각각 출력하세요.
정답 예시 코드 확인
Input Code 
import numpy as np
sales = np.arange(1,13).reshape(3,4)
idx = np.argmax(sales)
row, col = divmod(idx, 4)
print(row, col)
Output 
2 3

문제 99
99. product 와 cost 로부터 이익률(이익/원가 *100)을 perc 로 계산해 출력하세요.
정답 예시 코드 확인
Input Code 
import numpy as np
product = np.array([10,20,30,40])
cost = np.array([7,15,22,35])
profit = product - cost
perc = profit / cost * 100
print(perc)

[42.85714286 33.33333333 36.36363636 14.28571429]
