Rectangle(w, h) 클래스를 만들고 area(), perimeter() 메서드를 작성하세요. 각각 넓이와 둘레를 반환하세요.

class Rectangle:
    def __init__(self, w, h):
        self.w = w
        self.h = h

    def area(self):
        return self.w * self.h

    def perimeter(self):
        return 2 * (self.w + self.h)

위 BankAccount에 잔액 부족이면 출금하지 말고 잔액 부족을 출력한 뒤 현재 잔액을 반환하도록 수정하세요.

class BankAccount:
    def __init__(self, balance):
        self.balance = balance

    def withdraw(self, amount):
        if amount > self.balance:
            print('잔액 부족')
            return self.balance
        self.balance -= amount
        return self.balance

print(acc.withdraw(3000))

25번 문제

클래스메서드 — 대체 생성자 from_string
Point 클래스에 x,y를 저장하는 생성자를 만들고, 클래스메서드 from_string('3,4')으로 Point를 만드는 기능을 추가하세요.

힌트 / 출력
호출 예시
p = Point.from_string('3,4')
print(p.x, p.y)
출력 예시
3 4
정답 코드 보기
정답
class Point:
    def __init__(self, x, y):
        self.x = x
        self.y = y

    @classmethod
    def from_string(cls, s):
        a, b = s.split(',')
        return cls(int(a), int(b))

p = Point.from_string('3,4')
print(p.x, p.y)
