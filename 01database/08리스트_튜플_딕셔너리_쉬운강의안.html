리스트·딕셔너리·for문·JSON

리스트는 “번호가 붙은 사탕 상자”, 딕셔너리는 “이름표가 붙은 사탕 상자”로 비유해 구조를 이해하고, for문으로 순회하는 방법과 JSON 읽기 연습까지 진행한다.

학습 목표 1. 리스트(번호 사탕 상자) 2. 중첩 리스트(상자 안 상자) 3. 딕셔너리(이름표 사탕 상자) 4. 중첩 딕셔너리 5. 딕셔너리 안에 리스트 6. 복합 구조(딕셔너리+리스트 혼합) 7. for문으로 상자 순회 8. JSON 구조 읽기 9. JSON/for문 연습문제 정리

학습 목표

  • 리스트(List)와 딕셔너리(Dictionary)의 차이를 “사탕 상자” 비유로 설명할 수 있다.
  • 중첩 구조(리스트 안 리스트, 딕셔너리 안 딕셔너리, 딕셔너리+리스트 혼합)를 읽을 수 있다.
  • for문으로 리스트/딕셔너리/혼합 구조를 순회하는 기본 패턴을 작성할 수 있다.
  • JSON 모양(= 딕셔너리+리스트)을 보고 필요한 값에 접근하거나 전체를 출력할 수 있다.

1. 리스트(List) — 번호가 붙은 사탕 상자

리스트는 순서가 있는 자료 묶음이다. 각 요소에는 0부터 시작하는 번호(인덱스)가 자동으로 붙는다.

사탕 상자 비유
🍬 사탕 상자(리스트)
┌───────────────┐
│ 0번 🍓 딸기   │
│ 1번 🍇 포도   │
│ 2번 🍋 레몬   │
└───────────────┘
예시 코드
candies = ["딸기사탕", "포도사탕", "레몬사탕"]

print(candies[0])  # 0번 사탕 꺼내기
print(candies[2])  # 2번 사탕 꺼내기
  • 번호로 꺼낸다: candies[0]
  • 순서가 중요: 들어간 순서대로 번호가 매겨진다

2. 중첩 리스트 — 상자 안에 상자

리스트 안에 리스트가 들어 있으면, 큰 상자 안에 작은 상자가 있는 형태가 된다.

예시 코드
boxes = [
  ["딸기사탕", "포도사탕"],
  ["초코사탕", "카라멜"]
]
사탕 상자 비유
🍬 큰 사탕 상자(리스트)
┌─────────────────────┐
│ 0번 ┌───────────┐   │
│     │ 🍓 🍇    │   │
│     └───────────┘   │
│ 1번 ┌───────────┐   │
│     │ 🍫 🍬    │   │
│     └───────────┘   │
└─────────────────────┘
값 꺼내기
print(boxes[0])      # 첫 번째 작은 상자
print(boxes[0][1])   # 첫 번째 작은 상자의 두 번째 사탕(= 포도사탕)
  • 중첩 구조는 대괄호를 이어서 접근한다: boxes[0][1]

3. 딕셔너리(Dictionary) — 이름표가 붙은 사탕 상자

딕셔너리는 이름표(key)로 값을 찾는 상자다. 리스트처럼 번호가 자동으로 붙지 않는다.

사탕 상자 비유
🏷️ 이름표 사탕 상자(딕셔너리)
┌──────────────────┐
│ "빨강" → 🍓      │
│ "보라" → 🍇      │
│ "노랑" → 🍋      │
└──────────────────┘
예시 코드
candy_box = {
  "빨강": "딸기사탕",
  "보라": "포도사탕",
  "노랑": "레몬사탕"
}

print(candy_box["빨강"])
print(candy_box["노랑"])
  • 이름표(key)로 꺼낸다: candy_box["빨강"]
  • 순서보다 “이름표”가 중요

4. 중첩 딕셔너리 — 큰 이름 상자 안에 작은 이름 상자

딕셔너리 안에 딕셔너리가 들어가면, 이름표 상자 안에 이름표 상자가 된다.

예시 코드
store = {
  "A코너": {
    "빨강": "딸기사탕",
    "보라": "포도사탕"
  },
  "B코너": {
    "갈색": "초코사탕"
  }
}
값 꺼내기
print(store["A코너"]["보라"])  # 포도사탕
print(store["B코너"]["갈색"])  # 초코사탕
  • 중첩 딕셔너리는 키를 이어서 접근한다: store["A코너"]["보라"]

5. 딕셔너리 안에 리스트 — 이름 상자 안의 번호 상자

딕셔너리 값(value) 자리에 리스트가 들어가면, 이름표 상자 안에 번호 상자가 있는 형태다.

예시 코드
gift = {
  "어린이": ["딸기", "포도"],
  "어른": ["커피", "홍삼"]
}
값 꺼내기
print(gift["어린이"][0])  # 딸기
print(gift["어른"][1])    # 홍삼
  • gift["어린이"]는 리스트이므로, 그 다음엔 [0]처럼 번호로 접근한다.

6. 복합 구조 — 딕셔너리 + 리스트가 섞인 구조

실전에서는 딕셔너리와 리스트가 섞여 있는 구조를 자주 만난다(특히 API/JSON 응답).

예시 코드
shop = {
  "1층": [
    {
      "이름": "과일코너",
      "사탕": ["딸기", "포도"]
    },
    {
      "이름": "초코코너",
      "사탕": ["다크초코", "밀크초코"]
    }
  ]
}
끝판왕 접근 예시
print(shop["1층"][1]["사탕"][0])  # 다크초코
한 줄 해석
  • shop["1층"] : 1층에 있는 코너 목록(리스트)
  • [1] : 두 번째 코너(딕셔너리)
  • ["사탕"] : 그 코너의 사탕 목록(리스트)
  • [0] : 첫 번째 사탕

7. for문으로 상자들을 순회하는 법

핵심 규칙
  • 리스트 상자는 for item in list로 하나씩 꺼낸다.
  • 딕셔너리 상자는 for key in dict로 이름표를 하나씩 꺼낸다.
  • 안에 또 상자가 있으면, 그만큼 for문을 한 겹 더 만든다.

7-1. 리스트 순회

candies = ["딸기", "포도", "레몬"]

for candy in candies:
  print(candy)

7-2. 중첩 리스트 순회

boxes = [
  ["딸기", "포도"],
  ["초코", "카라멜"]
]

for box in boxes:
  for candy in box:
    print(candy)

7-3. 딕셔너리 순회

candy_box = {"빨강":"딸기", "보라":"포도", "노랑":"레몬"}

for color in candy_box:
  print(color, candy_box[color])

또는 .items()를 쓰면 (이름표, 사탕)을 같이 꺼낼 수 있다.

for color, candy in candy_box.items():
  print(color, candy)

7-4. 딕셔너리 안 리스트 순회

gift = {"어린이":["딸기","포도"], "어른":["커피","홍삼"]}

for person in gift:
  for candy in gift[person]:
    print(person, candy)

7-5. 복합 구조(shop) 순회

shop = {
  "1층": [
    {"이름":"과일코너", "사탕":["딸기","포도"]},
    {"이름":"초코코너", "사탕":["다크초코","밀크초코"]}
  ]
}

for floor in shop:
  for corner in shop[floor]:
    print("코너:", corner["이름"])
    for candy in corner["사탕"]:
      print(" -", candy)

8. JSON 구조 읽기

JSON은 모양이 파이썬의 딕셔너리 + 리스트와 거의 같다. 그래서 JSON을 읽는 것은 곧 “이름 상자와 번호 상자를 읽는 것”이다.

JSON 읽기 공식
  • { } → 딕셔너리(이름표 상자)
  • [ ] → 리스트(번호 상자)
  • "key": value → 이름표 : 내용물
예시(JSON 모양)
{
  "store": {
    "A": ["딸기", "포도"],
    "B": ["초코"]
  }
}

위 JSON은 파이썬으로 보면, 딕셔너리 안에 딕셔너리, 그 안에 리스트가 있는 구조다.

9. JSON/for문 연습문제

연습 1. 값 접근하기

아래 데이터에서 (1) 첫 번째 학생 이름, (2) 두 번째 학생의 첫 과목을 출력하시오.

data = {
  "반": [
    {"이름": "철수", "과목": ["수학", "영어"]},
    {"이름": "영희", "과목": ["국어"]}
  ]
}
힌트
  • "반"은 리스트다 → [0] 또는 [1]로 학생을 고른다.
  • 학생은 딕셔너리다 → ["이름"], ["과목"]으로 꺼낸다.
  • "과목"은 리스트다 → [0]로 첫 과목을 고른다.
정답 코드 보기 
print(data["반"][0]["이름"])
print(data["반"][1]["과목"][0])

연습 2. for문으로 전체 출력하기

모든 학생의 이름과 과목을 아래 형태로 출력하시오.

철수
 - 수학
 - 영어
영희
 - 국어
힌트

바깥은 data["반"](리스트), 안쪽은 학생 딕셔너리의 ["과목"](리스트)다.

정답 코드 보기 
for student in data["반"]:
  print(student["이름"])
  for subject in student["과목"]:
    print(" -", subject)

연습 3. 복합(shop) 구조 읽기

아래에서 “초코코너”의 첫 번째 사탕을 출력하시오.

shop = {
  "1층": [
    {"이름":"과일코너", "사탕":["딸기","포도"]},
    {"이름":"초코코너", "사탕":["다크초코","밀크초코"]}
  ]
}
힌트
  • "1층" → 리스트
  • [1] → 두 번째 코너(초코코너)
  • ["사탕"] → 사탕 리스트
  • [0] → 첫 번째 사탕
정답 코드 보기 
print(shop["1층"][1]["사탕"][0])

연습 4. 복합(shop) 전체 순회 출력

층 → 코너 이름 → 사탕 목록을 전부 출력하시오.

힌트

바깥은 딕셔너리(층), 안쪽은 리스트(코너들), 그 안은 딕셔너리(코너), 마지막은 리스트(사탕)다.

정답 코드 보기 
for floor in shop:
  print("==", floor, "==")
  for corner in shop[floor]:
    print("코너:", corner["이름"])
    for candy in corner["사탕"]:
      print(" -", candy)

정리

  • 리스트 = 번호가 붙은 사탕 상자 → [번호]
  • 딕셔너리 = 이름표가 붙은 사탕 상자 → ["키"]
  • 중첩 = 상자 안 상자 → [ ]와 [" "]를 순서대로 이어서 접근
  • for문 = 상자에서 하나씩 꺼내보는 직원 → 바깥 상자부터 차례로 순회
  • JSON = 딕셔너리+리스트 모양의 데이터 → 구조( { } / [ ] )만 읽으면 된다
마스터 공식
구조 읽기: 딕셔너리 → 리스트 → 딕셔너리 → 리스트
코드 작성: 바깥 상자부터 for문을 하나씩 추가
